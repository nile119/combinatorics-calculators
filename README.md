# combinatorics-calculators

Набор различных программ по элементарной теории вероятностей, написанных на Python 2.x/3.x.

| Программа | Описание | Используемые модули |
| --- | --- | --- |
| 1. Exponentiation | Консольная программа для возведения в степень целочисленных значений | string, random, os |
| 2. Combinatorics numbers | Программа подсчёта комбинаторных чисел (с GUI) | math, tkinter |
| 3. Functions | Программа построения некоторых классических распределений (с GUI)| math, tkinter, numpy, matplotlib |
